## 🎯 Quick Start Guide

### 1. **Environment Setup**
Create your `.env` file in the server directory:
```bash
# Essential API Keys
GEMINI_API_KEY=your_gemini_api_key_here
OPENAI_API_KEY=your_openai_api_key_here (optional)

# Database Configuration
MONGO_URI=mongodb://localhost:27017/imentorAI
NODE_ENV=development

# Server Configuration
PORT=4007
FRONTEND_PORT=4004

# Security
JWT_SECRET=your_super_secret_jwt_key
SESSION_SECRET=your_session_secret_here
```

### 2. **Launch Sequence**
```bash
# Install all dependencies
npm run install-all

# Start the full application
npm start

# Or run in development mode
npm run dev
```

### 3. **Access Your AI Assistant**
- 🌐 **Main Application**: http://localhost:4004
- ⚙️ **API Server**: http://localhost:4007
- 📊 **Health Check**: http://localhost:4007/health

## 🐳 Production Deployment

### **Docker Swarm (Recommended)**
```bash
# Initialize swarm
docker swarm init

# Deploy stack
docker stack deploy -c docker-compose.prod.yml imentorAI

# Scale services
docker service scale imentorAI_backend=3
```

### **PM2 Cluster Mode**
```bash
# Start with PM2
npm run pm2:start

# Monitor processes
npm run pm2:status

# View logs
npm run pm2:logs
```

### **Nginx Load Balancer**
```nginx
upstream imentorAI_backend {
    server localhost:4007;
    server localhost:4008;
    server localhost:4009;
}

server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        proxy_pass http://imentorAI_backend;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
